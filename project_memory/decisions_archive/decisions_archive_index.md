# decisions_archive_index.md

## Propósito del índice

Este índice registra los archivos históricos de decisiones archivadas dentro de `project_memory/decisions_archive/`.

Su función es mantener trazabilidad cuando una decisión activa deja de estar vigente o es sustituida.

No sustituye el contenido de cada archivo archivado.

No debe duplicar explicaciones largas.

---

## Criterio de orden

- orden principal: fecha descendente
- formato de nombre de archivo: `YYYY-MM-DD__topic__slug.md`
- id de archivo archivado: `archived-decision-YYYY-MM-DD-XX`

---

## Entradas

Actualmente no hay decisiones archivadas registradas en esta carpeta.

La primera entrada deberá añadirse cuando una decisión activa vigente sea sustituida, corregida o deje de aplicar.
