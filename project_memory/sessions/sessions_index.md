# sessions_index.md

## Propósito del índice

Este índice registra las entradas históricas de `project_memory/sessions/`.

Su función es localizar sesiones útiles sin convertir la carpeta en un bloque opaco o difícil de navegar.

No sustituye el contenido de las sesiones.

No debe contener resúmenes largos.

---

## Criterio de orden

- orden principal: fecha descendente
- formato de nombre de archivo: `YYYY-MM-DD__slug-del-bloque.md`
- id de sesión: `session-YYYY-MM-DD-XX`
