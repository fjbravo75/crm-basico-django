# CRM Básico con Django

CRM básico desarrollado con Django como proyecto de máster y portfolio.

## Estado del proyecto

Arranque inicial completado:
- repositorio local Git inicializado
- repositorio remoto en GitHub conectado
- proyecto base de Django creado

## Objetivo

Construir una aplicación CRM pequeña pero sólida, con:
- gestión de clientes
- seguimiento de interacciones
- búsqueda y filtrado simples
- estructura profesional para presentación en portfolio

## Stack tecnológico

- Python
- Django
- SQLite (desarrollo)
- Git y GitHub

## Demo local rápida

- Ejecuta `python manage.py migrate`
- Ejecuta `python manage.py seed_demo_crm`
- Arranca el servidor con `python manage.py runserver`
- Entra en `http://127.0.0.1:8000/acceso/login/`
- Usa el usuario `maria.ortega` y la contraseña `DemoCRM123!`
