# update_index.md

## Cuándo usar este workflow

Usa este workflow cuando se haya creado una nueva entrada en `project_memory/sessions/` o en `project_memory/decisions_archive/`.

---

## Pasos

1. Identifica qué índice corresponde actualizar.
2. Abre solo el índice necesario.
3. Actualiza `updated_at`.
4. Añade la nueva entrada en el orden definido.
5. Usa una descripción breve y funcional.
6. No conviertas el índice en resumen largo del contenido.

---

## No hagas esto

- no actualices índices si no se creó una nueva pieza histórica
- no metas texto narrativo largo
- no dupliques dentro del índice el cuerpo completo de la entrada
